Component implementation requirements:

1. **Position:** Fixed overlay, bottom-right corner (`fixed bottom-4 right-4 z-[9999]`). Dark translucent background (`bg-black/85`). Monospace font (`font-mono`). Text size `text-[11px]`. Max width ~320px.

2. **Collapsed state (D-01):** When `collapsed` is true, render only a single status line showing `phase` and progress percentage (`framesEncoded/totalFrames`). The line is clickable — calls `onToggleCollapse` on click (D-02). Use `cursor-pointer` on the collapsed bar.

3. **Expanded state:** Full panel per SPEC layout mockup:
   - Header row: "EXPORT METRICS" title + [x] button. The [x] calls `onToggleCollapse` (collapses, per D-01 — does NOT fully dismiss).
   - Section: Phase, Progress (framesEncoded / totalFrames), Encode FPS (encodeFps.toFixed(1)), Frame time (lastFrameMs.toFixed(1) ms), Avg frame (averageFrameMs.toFixed(1) ms), Elapsed (format as seconds with 1 decimal: `(elapsedMs/1000).toFixed(1) s`), ETA.
   - ETA row (D-04): **Omit the ETA row entirely** when `encodeFps <= 0`. Do NOT render the row at all — no "---", no placeholder. Only render the ETA row when `encodeFps > 0`, with value: `((totalFrames - framesEncoded) / encodeFps).toFixed(1) s`.
   - Separator line.
   - MEMORY section: JS Heap (`heapUsedMB.toFixed(0) / heapTotalMB.toFixed(0) MB`), Heap limit (`heapLimitMB.toFixed(0) MB`), Active bitmaps, Active frames, Null samples.
   - Chrome guard (D-10): If `heapUsedMB === 0 && heapTotalMB === 0 && heapLimitMB === 0`, display "N/A" for all three heap rows instead of "0 / 0 MB".
   - Separator line.
   - DEVICE section: RAM (`deviceMemoryGB` GB, or "N/A" if 0), CPU cores.
   - Decode setup row: `decodeSetupMs.toFixed(0) ms` (show once decodeSetupMs > 0).

4. **Guard for null metrics:** If `metrics` is null, render nothing (return null).

5. **Guard for visibility:** If `visible` is false, render nothing (return null).

6. **Styling:** Use Tailwind classes exclusively. Rows are `flex justify-between` with label on left (text-neutral-400) and value on right (text-neutral-200). Section headers in uppercase text-neutral-500 text-[10px]. [x] button: `text-neutral-500 hover:text-neutral-300`. Rounded corners `rounded-lg`. Padding `p-3`. Border `border border-white/10`.

7. **Transition (Claude's discretion):** Add `transition-all duration-200` on the container for smooth collapse/expand.

Export as named export: `export function ExportMetricsPanel(...)`.

**4. Add metrics polling effect (D-08: setInterval at 250ms, NOT rAF):**
```typescript
useEffect(() => {
  if (!METRICS_ENABLED) return;
  if (!isExporting) {
    // Stop polling when export ends — panel stays with final snapshot (D-06)
    if (metricsIntervalRef.current) {
      clearInterval(metricsIntervalRef.current);
      metricsIntervalRef.current = null;
    }
    // Set final snapshot so panel keeps showing last metrics
    if (metricsRef.current) {
      setMetricsSnapshot({ ...metricsRef.current });
    }
    return;
  }
  // Start polling
  metricsIntervalRef.current = setInterval(() => {
    if (metricsRef.current) {
      setMetricsSnapshot({ ...metricsRef.current });
    }
  }, 250);
  return () => {
    if (metricsIntervalRef.current) {
      clearInterval(metricsIntervalRef.current);
      metricsIntervalRef.current = null;
    }
  };
}, [isExporting]);
```

**5. Add Shift+M keyboard shortcut (D-12):**
```typescript
useEffect(() => {
  if (!METRICS_ENABLED) return;
  function handleKeyDown(e: KeyboardEvent) {

      setMetricsVisible(v => !v);
    }
  }
  document.addEventListener('keydown', handleKeyDown);
  return () => document.removeEventListener('keydown', handleKeyDown);
}, []);
```
