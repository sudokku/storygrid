## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-26-01 | Tampering | Google Fonts CDN response | accept | Google Fonts served over HTTPS with SRI not practical for dynamically-generated CSS; this is the standard pattern used by millions of sites. The existing preconnect tags already reference these domains. Risk: negligible -- Google CDN is a trusted origin. |
| T-26-02 | Information Disclosure | fontFamily string in overlay state | accept | fontFamily is a UI preference string (e.g. "Bebas Neue") stored in client-side Zustand state with no persistence. Contains no PII or sensitive data. |
| T-26-03 | Denial of Service | Google Fonts CDN unavailable | accept | font-display: swap ensures text remains visible in system fallback font if CDN is unreachable. No functionality is blocked -- only visual fidelity degrades gracefully. |
