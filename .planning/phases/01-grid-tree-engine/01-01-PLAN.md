  <action>
    1. Create `src/lib/tree.ts` implementing all functions. Follow the pure-function pattern from RESEARCH.md — no Immer, no side effects, spread + map recursion:

    **createLeaf()**: `import { nanoid } from 'nanoid';` returns `{ type: 'leaf', id: nanoid(), mediaId: null, fit: 'cover' }`.

    **findNode(root, id)**: Recursive DFS. If `root.id === id` return root. If container, iterate children recursively. Return null if not found.

    **findParent(root, childId)**: Recursive DFS. If container and any direct child has `id === childId`, return this container. Otherwise recurse into container children. Return null if not found or if childId is root.

    **getAllLeaves(root)**: If leaf, return `[root]`. If container, `root.children.flatMap(getAllLeaves)`.

    **splitNode(root, nodeId, direction)**: Three cases per RESEARCH.md Pattern 3:
    - CASE A (root is a leaf being split): return new container `{ type: 'container', id: nanoid(), direction, sizes: [1, 1], children: [root, createLeaf()] }`
    - CASE B (parent exists, parent.direction === direction — per D-03): append new leaf to parent.children, push `1` to parent.sizes. Use recursive mapNode to produce new tree.
    - CASE C (parent exists, parent.direction !== direction): replace leaf in parent with new container `{ type: 'container', id: nanoid(), direction, sizes: [1, 1], children: [leaf, createLeaf()] }`. Use recursive mapNode.

    Helper `mapNode(root, id, updater)`: Recursive function that returns new tree with node matching id replaced by updater(node). Uses spread for containers: `{ ...root, children: root.children.map(c => mapNode(c, id, updater)) }`.

    Helper `replaceNode(root, targetId, replacement)`: Uses mapNode with updater that ignores the old node and returns replacement.

    **mergeNode(root, containerId)**: Find the container. Replace it in the tree with its first child (preserving that child's type, id, mediaId if leaf).

    **removeNode(root, nodeId)**: Find parent of the node. Remove node from parent.children and corresponding index from parent.sizes. If parent left with 1 child, replace parent with that child in grandparent. If nodeId is the root's only content, return root unchanged.

    **resizeSiblings(root, containerId, index, delta)**: Find container by id. Create new sizes array: `newSizes[index] += delta`, `newSizes[index + 1] -= delta`. Clamp: if either value < 0.1, adjust both so neither goes below 0.1. Named constant `MIN_CELL_WEIGHT = 0.1`. Return new tree with container's sizes updated.

    **updateLeaf(root, nodeId, updates)**: Use mapNode. If target node is a leaf, return `{ ...node, ...updates }`. If not a leaf, return node unchanged.

    **buildInitialTree()**: Per D-06: `{ type: 'container', id: nanoid(), direction: 'vertical', sizes: [1, 1], children: [createLeaf(), createLeaf()] }`.
