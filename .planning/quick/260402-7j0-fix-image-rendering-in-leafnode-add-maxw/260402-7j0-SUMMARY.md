## What Was Built

Two targeted fixes to `src/Grid/LeafNode.tsx`:

1. **maxWidth override (FIX-IMG-RENDER):** In `renderMedia()`, the explicit-sizing `<img>` element's inline style object now includes `maxWidth: 'none'`. Tailwind Preflight sets `max-width: 100%` on all `img` elements globally. When an absolutely-positioned image is sized wider than its containing block (as cover mode requires for portrait images in landscape cells), Preflight's constraint clamps the rendered width below the intended `scaledW` value, distorting the aspect ratio. Setting `maxWidth: 'none'` removes this constraint.

2. **select-none (FIX-IMG-SELECT):** The outer cell `<div>`'s className now includes `select-none`. This applies `user-select: none` to all cell content, preventing users from accidentally selecting image pixels or placeholder text via click-and-drag.
