# BLOCKING CONSTRAINTS — Read Before Anything Else

> Each constraint was discovered through failure in Phase 25 (current DnD
> implementation). Acknowledge each one explicitly before writing any new
> DnD code in Phase 27+.

- [ ] CONSTRAINT: **Never run TouchSensor + MouseSensor simultaneously.** dnd-kit docs explicitly forbid this; it is the primary hypothesized cause of Phase 25's flaky swap + broken 5-zone detection. v1.5 MUST use a single PointerSensor.
- [ ] CONSTRAINT: **Never spread listener props before explicit pointer handlers on the same element.** On LeafNode, `{...dragListeners}` placed before `onPointerDown={handlePan}` let React overwrite dnd-kit's handler. Spreads go LAST.
- [ ] CONSTRAINT: **Drag state must NOT live in gridStore.** Ephemeral data in the main store pollutes undo/redo history. Use a separate vanilla Zustand `dragStore`.
- [ ] CONSTRAINT: **Never migrate two DnD engines in parallel.** Swap wholesale — remove the old `@dnd-kit` wiring in the SAME phase that adds new wiring (Phase 28).
- [ ] CONSTRAINT: **Canvas `<LeafNode>` previews cannot use `cloneNode`.** Canvas content doesn't serialize. Use `canvas.toDataURL()` → `<img>` for the drag ghost.

**Do not proceed with Phase 27 execution until all five boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Dual sensors | TouchSensor + MouseSensor both registered on DndContext | blocking | Single PointerSensor on the DndContext, full stop |
| Listener spread before explicit handler | `{...listeners} onPointerDown={x}` on same JSX element | blocking | Code review gate: spreads LAST in JSX |
| Drag state in gridStore | moveCell invoked optimistically during drag-over | blocking | dragStore is a separate vanilla zustand store; commit only on drop |
| 500ms hold delay | Collides with iOS 17.2+ image-action menu | blocking | 250ms + 5px tolerance |
| Parallel DnD engines during migration | Two engines mounted simultaneously | blocking | Phase 28 removes old wiring in the same phase that wires the new engine |
| `cloneNode` on canvas leaf preview | Ghost renders blank canvas | advisory | `toDataURL()` → img in DragPreviewPortal |
| transform: scale() coord math | Viewport↔canvas translations must divide by scale | advisory | Document in computeDropZone.ts; unit tests with scale≠1 |
