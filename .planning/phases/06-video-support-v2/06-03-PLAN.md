**Play/Pause button (per D-09, UI spec):**
- `w-11 h-11` (44px touch target), `rounded-full`, `flex items-center justify-center`.
- `hover:bg-muted`, `text-foreground`.
- Icon: `Play` or `Pause` from lucide-react, size 16.
- On click: toggle `isPlaying`. When setting to playing, iterate `videoElementRegistry.values()` and call `.play()` on each. When pausing, call `.pause()` on each. Use `useEditorStore.getState()` for the action calls.
- Aria label: "Play" or "Pause".

**Scrubber (per D-11, UI spec):**
- `<input type="range">` with `min={0}`, `max={totalDuration}`, `step={0.1}`, `value={playheadTime}`.
- Full width via `flex-1`.
- On change: call `seekAll(parseFloat(e.target.value))` then `setPlayheadTime(value)`.
- `seekAll` function: iterates `videoElementRegistry.values()`, sets `video.currentTime = time` on each. Then calls each draw function in `videoDrawRegistry` via `requestAnimationFrame` to redraw still frames at the seek position.
- Style the track and thumb using Tailwind arbitrary properties: `[&::-webkit-slider-runnable-track]:h-1 [&::-webkit-slider-runnable-track]:rounded-full [&::-webkit-slider-runnable-track]:bg-muted [&::-webkit-slider-thumb]:w-4 [&::-webkit-slider-thumb]:h-4 [&::-webkit-slider-thumb]:rounded-full [&::-webkit-slider-thumb]:bg-[#3b82f6] [&::-webkit-slider-thumb]:appearance-none [&::-webkit-slider-thumb]:-mt-1.5`. Also add Firefox equivalents: `[&::-moz-range-track]` and `[&::-moz-range-thumb]` with equivalent styles.

**Time display (per D-09, UI spec):**
- Format: `M:SS / M:SS` (e.g., `0:03 / 0:08`).
- `text-muted-foreground text-xs tabular-nums` (tabular-nums for fixed-width digits).
- `min-w-[5rem]` to prevent layout shift.
- `text-right` alignment.
- Helper function `formatTime(seconds: number): string`:
  ```typescript
  function formatTime(s: number): string {
    const mins = Math.floor(s / 60);
    const secs = Math.floor(s % 60);
    return `${mins}:${String(secs).padStart(2, '0')}`;
  }
  ```

**Playhead update loop (per research Section 6):**
While `isPlaying`, use a `setInterval` at 100ms (10fps update) to read `currentTime` from the first video in `videoElementRegistry` and sync to `setPlayheadTime`. Also check if playback has reached `totalDuration` — if so, pause all and set `isPlaying(false)`.

```typescript
useEffect(() => {
  if (!isPlaying) return;
  const id = setInterval(() => {
    const [firstVideo] = videoElementRegistry.values();
    if (firstVideo) {
      const time = firstVideo.currentTime;
      setPlayheadTime(time);
      if (time >= totalDuration - 0.05) {
        // Playback complete — pause all
        for (const video of videoElementRegistry.values()) video.pause();
        setIsPlaying(false);
      }
    }
  }, 100);
  return () => clearInterval(id);
}, [isPlaying, totalDuration, setPlayheadTime, setIsPlaying]);
```
