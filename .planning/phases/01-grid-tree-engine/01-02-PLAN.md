    Store implementation using `create<GridStoreState>()(immer((set, get) => ({ ... })))`:

    - Initialize: `root: buildInitialTree()`, `mediaRegistry: {}`, `history: [{ root: structuredClone(buildInitialTree()) }]` (per RESEARCH: push initial state as history[0]), `historyIndex: 0`.
    - NOTE: The initial tree built by `buildInitialTree()` and the one in `history[0]` must be separate clones (not same reference). Call `buildInitialTree()` once, store it, then `structuredClone` it for history[0].

    - Every mutating action (split, merge, remove, resize, setMedia, updateCell) follows this pattern inside `set(state => { ... })`:
      ```
      // 1. Snapshot BEFORE mutation
      const plainRoot = current(state.root);
      const snap = structuredClone({ root: plainRoot });
      // 2. Clear redo stack
      state.history = state.history.slice(0, state.historyIndex + 1);
      // 3. Push snapshot
      state.history.push(snap);
      // 4. Cap at 50
      if (state.history.length > 50) {
        state.history.shift();
      } else {
        state.historyIndex = state.history.length - 1;
      }
      // 5. Apply mutation
      state.root = pureFunction(current(state.root), ...args);
      ```
      NOTE on cap logic: After `shift()`, historyIndex should be `state.history.length - 1` (not incremented). Consolidate: always set `state.historyIndex = state.history.length - 1` after push (whether shifted or not).

    - `setMedia(nodeId, mediaId)`: calls `updateLeaf(current(state.root), nodeId, { mediaId })` with history snapshot.
    - `addMedia(mediaId, dataUri)`: `state.mediaRegistry[mediaId] = dataUri` — NO history snapshot (registry is not tracked).
    - `removeMedia(mediaId)`: `delete state.mediaRegistry[mediaId]` — NO history snapshot.
    - `undo()`: if `state.historyIndex <= 0` return (no-op). Decrement `state.historyIndex`. Set `state.root = structuredClone(state.history[state.historyIndex].root)`.
    - `redo()`: if `state.historyIndex >= state.history.length - 1` return (no-op). Increment `state.historyIndex`. Set `state.root = structuredClone(state.history[state.historyIndex].root)`.
