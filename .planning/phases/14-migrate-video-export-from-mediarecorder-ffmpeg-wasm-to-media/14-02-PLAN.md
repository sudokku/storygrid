    Completely rewrite `src/lib/videoExport.ts`. Keep these existing functions UNCHANGED (they are reusable):
    - `computeLoopedTime` — pure helper, no changes needed

    - `destroyExportVideoElements` — cleanup helper, keep as-is
    - `hasAudioEnabledVideoLeaf` — decision helper, keep as-is

    DELETE these functions (replaced by new pipeline):
    - `buildAudioGraph` — was MediaRecorder-specific (uses MediaElementAudioSourceNode + MediaStreamAudioDestinationNode)

    ADD these new functions:

    **1. `seekAllVideosTo(timeSeconds, exportVideoElements)` (D-13)**
    - For each video in exportVideoElements, compute looped time via computeLoopedTime(timeSeconds, video.duration), then seek to that looped time
    - Use `seeked` event listener with 500ms setTimeout fallback
    - Early exit if already at target time (Math.abs diff < 0.01)
    - Return Promise.all of all seek promises

    **2. `collectAudioEnabledMediaIds(leaves, mediaTypeMap)` helper**
    - Returns Set of unique mediaIds where leaf.audioEnabled===true AND mediaTypeMap[mediaId]==='video'
    - Used to determine which cells to decode audio from

    **3. `mixAudioForExport(mediaRegistry, audioEnabledMediaIds, totalDurationSeconds)` (D-02, AUD-05, AUD-07)**
    - If audioEnabledMediaIds is empty, return null
    - SAMPLE_RATE = 48000, NUM_CHANNELS = 2
    - Create a temporary AudioContext({ sampleRate: 48000 }) for decoding
    - For each mediaId in audioEnabledMediaIds:
      - fetch(mediaRegistry[mediaId]) -> arrayBuffer -> tempCtx.decodeAudioData(arrayBuffer)
      - Wrap in try/catch — skip cells that fail decode (Pitfall 3: video cells without audio tracks)
      - Check audioBuffer.numberOfChannels > 0 before using
    - Close tempCtx after all decoding
    - Create OfflineAudioContext(NUM_CHANNELS, Math.ceil(totalDurationSeconds * SAMPLE_RATE), SAMPLE_RATE)
    - For each decoded buffer, schedule looped playback:
      - let offset = 0; while (offset < totalDurationSeconds) { create AudioBufferSourceNode, connect to destination, start(offset), stop(Math.min(offset + buffer.duration, totalDurationSeconds)), offset += buffer.duration }
      - Guard: if buffer.duration <= 0, break (prevent infinite loop)
    - Return offlineCtx.startRendering()

    **4. Rewrite `exportVideoGrid(root, mediaRegistry, mediaTypeMap, settings, totalDuration, onProgress)` (D-01, D-03, D-06, D-11, D-12, D-13, D-14)**

    The function signature changes: `onProgress` type becomes `(stage: 'preparing' | 'encoding', percent?: number) => void` — remove 'transcoding'.

    Implementation flow:
    a. VideoEncoder guard (D-06): if (typeof VideoEncoder === 'undefined') throw new Error('Video export requires WebCodecs. Please use Chrome 94+ or Firefox 130+.')
    b. onProgress('preparing')
    c. Build export video elements via buildExportVideoElements()
    d. Create stable 1080x1920 canvas + get 2d context (D-11)
    e. Codec selection (D-12): const isFirefox = navigator.userAgent.includes('Firefox'); const videoCodec = isFirefox ? 'vp9' : 'avc';
    f. Create BufferTarget, Output({ format: new Mp4OutputFormat(), target })
    g. Create CanvasSource(stableCanvas, { codec: videoCodec, bitrate: QUALITY_HIGH })
    h. output.addVideoTrack(videoSource)
    i. Audio setup (D-01, D-03): check hasAudioEnabledVideoLeaf. If true:
       - try { if (!(await canEncodeAudio('aac'))) registerAacEncoder(); }
       - if (await canEncodeAudio('aac')): create AudioBufferSource({ codec: 'aac', bitrate: 128_000 }), output.addAudioTrack(audioSource)
       - catch: log warning, set audioSource = null (D-03 non-fatal)
       - If canEncodeAudio('aac') still false after polyfill: log "Audio not supported in this browser — exporting video only", set audioSource = null
    j. await output.start() — MUST be after all addTrack calls (Pitfall 5)
    k. Read overlay state: useOverlayStore.getState()
    l. await document.fonts.ready (if available)
    m. Image cache: new Map for images, new Map for overlay images
    n. Frame encoding loop:
       - const FPS = 30; const totalFrames = Math.ceil(totalDuration * FPS);
       - for (let i = 0; i < totalFrames; i++):
         - await seekAllVideosTo(i / FPS, exportVideoElements) — handles looping internally
         - await renderGridIntoContext(stableCtx, root, mediaRegistry, 1080, 1920, settings, exportVideoElements, imageCache)
         - await drawOverlaysToCanvas(stableCtx, overlayState.overlays, overlayState.stickerRegistry, overlayImageCache, true)
         - await videoSource.add(i / FPS, 1 / FPS) — timestamps in SECONDS (Mediabunny high-level API)
         - onProgress('encoding', Math.round((i / totalFrames) * 100))
    o. After video loop — audio mix + encode (D-01, D-02):
       - if (audioSource):
         - try:
           - const audioEnabledIds = collectAudioEnabledMediaIds(getAllLeaves(root), mediaTypeMap)
           - const mixedBuffer = await mixAudioForExport(mediaRegistry, audioEnabledIds, totalDuration)
           - if (mixedBuffer) await audioSource.add(mixedBuffer)
         - catch: console.warn, D-03 non-fatal
    p. await output.finalize()
    q. destroyExportVideoElements(exportVideoElements)
    r. return new Blob([target.buffer as ArrayBuffer], { type: 'video/mp4' })

    Wrap the entire body in try/finally so destroyExportVideoElements always runs on error.

    IMPORTANT: Remove ALL MediaRecorder-related code. Remove ALL captureStream code. Remove ALL ffmpeg/transcodeToMp4 references. The new implementation has ZERO references to MediaRecorder, captureStream, transcodeWebmToMp4, or @ffmpeg/*.
