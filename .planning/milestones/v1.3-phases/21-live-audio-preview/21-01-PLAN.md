Create `src/hooks/useAudioMix.ts` implementing the full Web Audio API hook per decisions D-01 through D-05.

**Exports:** `export function useAudioMix(): { startAudio: () => void; stopAudio: () => void }`

**Internal refs (useRef):**
- `audioCtxRef = useRef<AudioContext | null>(null)` — per D-02, created once, never closed during component lifetime
- `nodeMapRef = useRef<Map<string, { sourceNode: MediaElementAudioSourceNode; gainNode: GainNode }>>(new Map())` — per D-03, never recreated

**`buildNodeGraph(ctx: AudioContext)` internal function:**
- Iterate `videoElementRegistry.entries()`.
- For each `[nodeId, videoEl]`: if `nodeMapRef.current.has(nodeId)` skip (idempotent per D-03/SC5).
- Otherwise: `const sourceNode = ctx.createMediaElementSource(videoEl)`, `const gainNode = ctx.createGain()`, `sourceNode.connect(gainNode)`, `gainNode.connect(ctx.destination)`.
- Store `{ sourceNode, gainNode }` in `nodeMapRef.current.set(nodeId, ...)`.

**`updateGains()` internal function:**
- Read `const { root } = useGridStore.getState()`.
- Call `getAllLeaves(root)` from `src/lib/tree.ts`.
- Compute `activeIds = new Set(leaves.filter(l => l.audioEnabled && l.hasAudioTrack && videoElementRegistry.has(l.id)).map(l => l.id))` — per D-04.
- Compute `gain = activeIds.size > 0 ? 1 / activeIds.size : 0`.
- For each `[nodeId, { gainNode }]` in `nodeMapRef.current`: set `gainNode.gain.value = activeIds.has(nodeId) ? gain : 0`.
- Stale entries (nodeId not in `videoElementRegistry`) also get `gain.value = 0` (Pitfall 3 from RESEARCH).

**`startAudio()` function (per D-02, D-06):**
- If `!audioCtxRef.current`: `audioCtxRef.current = new AudioContext()` — synchronous, in click callstack.
- Call `buildNodeGraph(audioCtxRef.current)` — idempotent, picks up new videos (Pitfall 4).
- Call `updateGains()`.
- If `audioCtxRef.current.state === 'suspended'`: `audioCtxRef.current.resume()` — guard prevents error if already running.

**`stopAudio()` function (per D-02):**
- If `audioCtxRef.current && audioCtxRef.current.state === 'running'`: `audioCtxRef.current.suspend()`.

**Reactive subscription (per D-05):**
- `useEffect` that subscribes to `useGridStore.subscribe((state, prev) => { ... })`.
- Inside subscriber: if `audioCtxRef.current` is null or `audioCtxRef.current.state !== 'running'`, return early.
- Compare `getAllLeaves(state.root)` vs `getAllLeaves(prev.root)` for any `audioEnabled` or `hasAudioTrack` change.
- If changed: call `updateGains()`.
- Return `unsub` from effect cleanup.

**Cleanup effect:**
- `useEffect(() => () => { audioCtxRef.current?.close(); nodeMapRef.current.clear(); }, [])` — Pitfall 6 from RESEARCH.

**Return:** `{ startAudio, stopAudio }`.

**Imports needed:**
- `import { useRef, useEffect, useCallback } from 'react'`
- `import { videoElementRegistry } from '../lib/videoRegistry'`
- `import { useGridStore } from '../store/gridStore'`
- `import { getAllLeaves } from '../lib/tree'`

```typescript
function handlePlayPause() {
  const store = useEditorStore.getState();
  if (store.isPlaying) {
    // Pause all
    for (const video of videoElementRegistry.values()) video.pause();
    stopAudio();
    store.setIsPlaying(false);
  } else {
    // startAudio MUST be first — AudioContext construction requires
    // synchronous user gesture callstack (D-06, LAUD-04)
    startAudio();
    for (const video of videoElementRegistry.values()) video.play();
    store.setIsPlaying(true);
  }
}
```

Add `stopAudio()` call when playback reaches the end. The current code at lines 47-51 is:
```typescript
if (time >= totalDuration - 0.05) {
  for (const video of videoElementRegistry.values()) video.pause();
```

Change to:
```typescript
if (time >= totalDuration - 0.05) {
  for (const video of videoElementRegistry.values()) video.pause();
```

Note: `stopAudio` must be accessible inside the useEffect. Since `useAudioMix` returns stable references (useCallback or refs internally), `stopAudio` can be added to the effect dependency array. Alternatively, capture `stopAudio` in a ref to avoid effect dependency — follow the same pattern PlaybackTimeline already uses with `useEditorStore.getState()` for avoiding stale closures. The simplest approach: since `stopAudio` only calls `audioCtxRef.current?.suspend()` which reads from a ref, the function identity does not need to be stable in deps — it can be called from inside the interval without being in the dep array, as long as the hook's internal ref is current. Use `useCallback` with empty deps in the hook to guarantee stable identity, then include `stopAudio` in the useEffect deps.
