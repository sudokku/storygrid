From src/store/gridStore.ts (lines 473-494 — MUST remain byte-identical after Phase 27):
```typescript
// Phase 9 D-04/D-05: atomic move of a leaf to a target cell's edge.
// edge === 'center' delegates to swapLeafContent (existing swap semantics).
// Other edges trigger a structural move via moveLeafToEdge (Plan 01 primitive).
// Single pushSnapshot per successful call = one undo entry.
moveCell: (fromId, toId, edge) =>
  set(state => {
    // No-op guards (D-04 / EC-05). These do NOT push a snapshot.
    if (fromId === toId) return;                            // ← Guard 1
    const src = findNode(current(state.root), fromId);
    if (!src || src.type !== 'leaf') return;                // ← Guard 2 (source not found OR not a leaf)
    const tgt = findNode(current(state.root), toId);
    if (!tgt || tgt.type !== 'leaf') return;                // ← Guard 3 (target not found OR not a leaf)

    pushSnapshot(state);
    if (edge === 'center') {
      state.root = swapLeafContent(current(state.root), fromId, toId);
    } else {
      state.root = moveLeafToEdge(current(state.root), fromId, toId, edge);
      useEditorStore.getState().setSelectedNode(null);
    }
  }),
```

Note on guard grouping: The source code collapses "node not found" and "not a leaf" into a single early-return for each of source and target — so there are TWO early-return statements (guard 2, guard 3), but FOUR distinct failure conditions, each of which must be individually tested:
  - Guard 1: `fromId === toId`
  - Guard 2a: `!src` (source node id does not exist in tree)
  - Guard 2b: `src.type !== 'leaf'` (source id refers to a container)
  - Guard 3a: `!tgt` (target node id does not exist in tree)
  - Guard 3b: `tgt.type !== 'leaf'` (target id refers to a container)

That is 5 distinct test scenarios. The ROADMAP details describe "four no-op guards" collapsing 2a/2b and 3a/3b into "source not found / target not found" categories — the orchestrator accepts either framing; this plan goes with the fuller five-scenario count for bulletproof coverage.
