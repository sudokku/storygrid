## Animation Contract

All values sourced from CONTEXT.md decisions. These are the only motion values the executor may use.

### Ghost Opacity (D-05, overrides GHOST-03)
- **Value:** `opacity: 0.2` (20%)
- **Rationale:** Lower opacity reveals the drop-zone indicators beneath the ghost.
- Applied on: `<img>` in `DragPreviewPortal.tsx`.

### Ghost Size Cap (D-03, D-04)
- **Cap value:** `200px` on both axes — `max-width: 200px; max-height: 200px`.
- **Implementation:** Tailwind arbitrary values `max-w-[200px] max-h-[200px]` on the `<img>` in `DragPreviewPortal.tsx` (or a CSS variable `--ghost-cap: 200px` so the value can be tuned without code changes — preferred).
- Note: exact cap value is flagged as "user unsure about 200px" in CONTEXT.md. Use CSS variable `--ghost-cap: 200px` in `src/index.css :root {}` and reference via `max-w-[var(--ghost-cap)] max-h-[var(--ghost-cap)]` so it can be tuned without a code change.

### Snap-Back Animation (D-07, CANCEL-02)
- **Duration:** 200ms
- **Easing:** `ease-in` (maps to `cubic-bezier(0.4, 0, 1, 1)`)
- **Requirement spec says:** `cubic-bezier(0.15, 1.0, 0.3, 1.0)` (CANCEL-02). CONTEXT.md D-07 says `ease-in`. Use CONTEXT.md value — `ease-in` — (deviates from CANCEL-02 easing curve — user decision in D-07).
- Applied on: `DragOverlay dropAnimation` config in `DragPreviewPortal.tsx`. Currently `null`; change to `{ duration: 200, easing: 'ease-in' }`.
- Fires on: `onDragCancel`, and also briefly on `onDragEnd` before `dragStore.end()` hides the ghost — acceptable per D-07.

### Drop Flash (D-08, DROP-08)
- **Duration:** 700ms total (per DROP-08 "Atlassian largeDurationMs")
- **Visual:** Accent-color background flash OR ring/outline flash — executor's choice (CONTEXT.md "Claude's discretion").
  - **Recommended:** `ring-2 ring-accent` momentarily applied, then a CSS transition fades it out. A `bg-accent/20` background overlay is the alternative.
  - **Recommended CSS:** `@keyframes drop-flash { 0% { box-shadow: 0 0 0 2px var(--ring); } 100% { box-shadow: none; } }` over 700ms `ease-out`.
  - Applied to: `LeafNode` wrapper `div` when `lastDropId === cellId` in `dragStore`.
- **Applies to:** All successful drops — center (swap) and all 4 edge zones. Source: CONTEXT.md "Claude's discretion" — yes, flash on all.

### Drag-Start Wobble (D-09, DRAG-05)
- **Rotation range:** ±1.5deg
- **Duration:** 150ms
- **Keyframes:**
  ```css
  @keyframes cell-wobble {
    0%   { transform: rotate(0deg); }
    25%  { transform: rotate(1.5deg); }
    75%  { transform: rotate(-1.5deg); }
    100% { transform: rotate(0deg); }
  }
  ```
- **Fill mode:** `animation-fill-mode: none` (return to 0 — standard behavior per CONTEXT.md "Claude's discretion").
- Applied on: source cell's wrapper `div` when `isDragging` is true. Class removed when `isDragging` returns false.
- Define as: `animate-cell-wobble` Tailwind extension in `tailwind.config.js` OR as global `@keyframes` in `src/index.css` referenced via `style={{ animation: 'cell-wobble 150ms ease-in-out' }}`.
- **Timing note:** 150ms is within the 150–300ms micro-interaction window from UI-INTELLIGENCE.md. Compliant.

### prefers-reduced-motion
- All three animations (snap-back, drop flash, drag wobble) must be suppressed when `prefers-reduced-motion: reduce` is active. Source: UI-INTELLIGENCE.md (HIGH severity).
  ```css
  @media (prefers-reduced-motion: reduce) {
    .animate-cell-wobble { animation: none; }
    /* drop-flash and snap-back: set duration to 0ms */
  }
  ```

## Interaction States

### Cell during active drag (source cell)
- `opacity: 0.4` on source cell (`sourceId === cellId`) — GHOST-07
- Wobble animation: `cell-wobble 150ms ease-in-out` fires once on drag activation
- `cursor: grabbing` on `document.body` — DRAG-02

### Ghost (DragOverlay)
- `opacity: 0.2` — D-05
- `max-width: var(--ghost-cap); max-height: var(--ghost-cap)` — D-03/D-04
- No rotation, no shadow added in Phase 29 (DRAG-06 lift shadow was Phase 28 scope)

### Drop target (cell being hovered)
- 2px accent-color ring on the cell wrapper — DROP-04 (already implemented in Phase 28; verify)
- Active zone icon: `text-white scale-[1.1] drop-shadow` — DROP-02
- Inactive zone icons: `text-white/30` — DROP-03 / D-10

### Post-drop flash (landed cell)
- `drop-flash 700ms ease-out` fires once
- Visible when `lastDropId === cellId` in `dragStore`
- `setTimeout(() => dragStore.clearLastDrop(), 700)` in `handleDragEnd`

### ESC cancel (keyboard)
- `KeyboardSensor` from `@dnd-kit/core` added to `useSensors`
- dnd-kit fires `onDragCancel` automatically on ESC
- `dropAnimation` (snap-back 200ms ease-in) fires on cancel
- `dragStore.end()` called after snap-back completes
