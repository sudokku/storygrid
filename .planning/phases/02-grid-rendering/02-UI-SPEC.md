## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-icon gap inside action bar, divider handle padding |
| sm | 8px | Action bar button padding, tooltip offset |
| md | 16px | Cell empty-state inner padding, canvas area padding |
| lg | 24px | Section padding in sidebar placeholder |
| xl | 32px | Canvas area min padding from viewport edge |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions:
- Divider hit area: 10px transparent zone (5px each side of the 2px line) — sized for comfortable grab without occluding cell content. Source: CONTEXT.md §Claude's Discretion.
- Touch/pointer target minimum for action bar buttons: 32px × 32px (icon 16px + 8px padding each side).
- Safe zone guide: 250px from top and bottom edges of the 1080×1920 canvas — driven by `--safe-zone-top` / `--safe-zone-bottom` CSS variables already defined in `src/index.css`. Not an 8-point multiple; it is a canvas domain value, not a UI spacing token.

## Component Inventory

Components introduced in Phase 2. All live in `src/Grid/`.

| Component | File | Description |
|-----------|------|-------------|
| GridNode | `src/Grid/GridNode.tsx` | Dispatcher — renders ContainerNode or LeafNode based on `node.type`. React.memo'd. |
| ContainerNode | `src/Grid/ContainerNode.tsx` | Flex row or column; renders children + Dividers between each. React.memo'd per-node slice. |
| LeafNode | `src/Grid/LeafNode.tsx` | Renders empty state, media state, selection border, action bar on hover. React.memo'd per-node slice. `isolation: isolate` for Safari. |
| Divider | `src/Grid/Divider.tsx` | 2px line with 10px hit area; setPointerCapture drag; local state only during drag; commits resize on pointerup. |
| ActionBar | `src/Grid/ActionBar.tsx` | Icon-only floating bar (Split H, Split V, Remove, Toggle Fit). Renders inside LeafNode, position: absolute, top-center. Uses shadcn Tooltip on each button. |
| CanvasWrapper | `src/Grid/CanvasWrapper.tsx` | 1080×1920px inner div + CSS transform scale via ResizeObserver. Replaces placeholder in CanvasArea.tsx. |
| SafeZoneOverlay | `src/Grid/SafeZoneOverlay.tsx` | Dashed guide lines at `--safe-zone-top` / `--safe-zone-bottom`. Conditionally rendered when `editorStore.showSafeZone` is true. |

shadcn component used: `Tooltip` (wraps Radix `@radix-ui/react-tooltip`). Added via `npx shadcn add tooltip` after init.

## Interaction States

### Leaf Cell States

| State | Visual |
|-------|--------|
| Empty, unselected, unhovered | `#1c1c1c` fill, `#333333` dashed border 1px, no action bar |
| Empty, hovered | Above + action bar fades in (150ms delay, 150ms transition) |
| Empty, selected | `#3b82f6` solid border 2px; no action bar visible unless also hovered |
| Filled, unselected, unhovered | Image renders with `object-fit` from `leaf.fit`; no overlay |
| Filled, hovered | Action bar fades in (150ms delay, 150ms fade); image dims to 85% brightness |
| Filled, selected | `#3b82f6` solid border 2px; action bar visible when also hovered |

### Divider States

| State | Visual |
|-------|--------|
| Canvas not hovered | Dividers invisible (opacity-0) |
| Canvas hovered (any part) | All dividers show as 2px `#444444` line (opacity-100, group-hover pattern) |
| Specific divider hovered | Grab handle appears: 24×6px pill shape in `#888888`, centered on divider; cursor changes to `col-resize` (vertical) or `row-resize` (horizontal) |
| Dragging | Cursor locked to `col-resize` / `row-resize` via setPointerCapture; siblings resize live; no store writes until pointerup |

### Canvas Scaling

- CanvasWrapper inner div: always 1080×1920px, never scaled directly
- `transform: scale({factor})` applied to outer wrapper only
- Scale factor = `(containerHeight * 0.90) / 1920`, recalculated on ResizeObserver callback
- Debounce ResizeObserver callback: 50ms trailing (prevents layout thrash on resize)
- `editorStore.zoom` multiplies on top of auto-fit factor: `finalScale = autoFitScale * zoom`

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA (action bar) | "Split Horizontal" / "Split Vertical" / "Remove" / "Toggle Fit" (tooltip labels) |
| Empty state heading | none — icon only (upload icon from lucide-react, centered in cell) |
| Empty state body | "Drop image or click to upload" (14px, `#666666`, below icon) |
| Empty state — Phase 2 note | Upload interaction wired in Phase 3; text is present but click does nothing in Phase 2 |
| Error state | Not applicable in Phase 2 — no user-triggered async operations |
| Destructive action: Remove Cell | No confirmation dialog — action is immediately undoable via Ctrl+Z. Tooltip label: "Remove cell" |
| Safe zone toggle label | "Safe Zone" (toolbar button tooltip, Phase 3 toolbar — safe zone overlay is wired in Phase 2) |

Note: "Remove" is an immediately reversible action (undo/redo via gridStore). No confirmation modal is required. Source: CONTEXT.md D-03 (undo is clean per-drag) + GRID-12 (undo/redo history).

## Interaction Contracts

### Action Bar Buttons (Phase 2)

| Button | Icon (lucide-react) | Tooltip Text | Action |
|--------|---------------------|--------------|--------|
| Split Horizontal | `SplitSquareHorizontal` | "Split horizontal" | `gridStore.split(nodeId, 'horizontal')` |
| Split Vertical | `SplitSquareVertical` | "Split vertical" | `gridStore.split(nodeId, 'vertical')` |
| Remove | `Trash2` | "Remove cell" | `gridStore.remove(nodeId)` |
| Toggle Fit | `Maximize2` (cover) / `Minimize2` (contain) | "Switch to contain" / "Switch to cover" | `gridStore.updateCell(nodeId, { fit: toggle })` |

### Deselect Gestures

1. Click canvas background (CanvasArea `onClick` — only fires if target is the wrapper itself, not a child)
2. Click already-selected leaf again (toggle: `setSelectedNodeId(null)`)

Source: CONTEXT.md D-12.

### Pointer Events — Divider Drag

```
pointerdown → setPointerCapture(e.pointerId) → start local state { startPos, startSizes }
pointermove → calculate delta → update local state → pass new sizes to ContainerNode via callback (no store write)
pointerup → releasePointerCapture → call gridStore.resize(containerId, newSizes) → clear local state
```

No dnd-kit involvement. Source: CONTEXT.md D-03, STATE.md §Decisions.

## Accessibility Baseline

- All action bar buttons: `aria-label` matching tooltip text (required since buttons are icon-only)
- Divider: `role="separator"` + `aria-orientation="vertical"` / `"horizontal"` + `aria-label="Resize divider"`
- Selected leaf: `aria-selected="true"` on the leaf wrapper element
- Empty leaf: `aria-label="Empty cell — drop image or click to upload"`
- Tooltip trigger: Radix Tooltip handles `aria-describedby` automatically
