## EffectsPanel internals

- **Combined selector** (single subscription) via `useShallow`:
  ```tsx
  const { effects, hasMedia } = useGridStore(
    useShallow((s) => {
      const n = findNode(s.root, nodeId);
      const leaf = n && n.type === 'leaf' ? n : null;
      return {
        effects: leaf?.effects ?? DEFAULT_EFFECTS,
        hasMedia: leaf?.mediaId != null,
      };
    }),
  );
  ```
  Verified by acceptance criterion `[ "$(grep -c 'useGridStore(' EffectsPanel.tsx)" -eq 1 ]` — exactly one subscription call. Action dispatch goes through imperative `useGridStore.getState().<action>(...)` which does not subscribe.
- **Preset chip strip** — 6 buttons in a horizontal scroll container, each rendering a 48×48 thumbnail with a `ring-2 ring-[#3b82f6]` ring when active.
- **Slider rows** — 4 rows generated by an inline `renderSlider(field, label)` helper to avoid 4× duplication. Brightness/contrast/saturation use `min={-100} max={100}`; blur uses `min={0} max={20}`. Value readout shows `+50` / `-20` / `0` for the signed sliders and `5px` for blur.
- **Drag semantics**:
  - `onPointerDown` → `isDraggingRef.current = true` + `beginEffectsDrag(nodeId)` (one snapshot).
  - `onChange` → `setEffects(nodeId, { [field]: parseInt(value) })` (no snapshot).
  - `onPointerUp` / `onPointerCancel` → `isDraggingRef.current = false`.
  - Keyboard fallback: `onKeyDown` recognizes 8 nudge keys and calls `beginEffectsDrag` if `!isDraggingRef.current`. Each keystroke produces one snapshot.
- **Disabled state** — when `!hasMedia`, the body is wrapped in `opacity-40 pointer-events-none` and a `Add media to apply effects` line appears above. The DOM still contains the chips/sliders so they remain testable.

## Tests added

`src/Editor/__tests__/EffectsPanel.test.tsx` — 8 tests covering:
1. All 6 chips render with correct display names (`B&W`, `Sepia`, `Vivid`, `Fade`, `Warm`, `Cool`).
2. All 4 sliders render with correct ARIA labels.
3. Clicking the B&W chip applies `preset: 'bw', saturation: -100`.
4. Clicking Reset effects restores `DEFAULT_EFFECTS`.
5. Clicking Reset cell restores effects + zeroes pan/scale.
6. Slider drag (pointerDown + 2× change + pointerUp) commits exactly **one** history entry; undo restores brightness to 0.
7. D-15 — slider change after applying a preset clears the preset flag.
8. Disabled state — leaf without media renders the `Add media to apply effects` notice.

## Pre-existing test fix

`src/test/phase07-02-sidebar-thumbnail.test.tsx` previously asserted "no img exists in the panel" for cells without thumbnails using `document.querySelector('img')`. Phase 11's chip strip now renders 6 chip imgs in the panel, so those queries broke. Fix scopes all 4 queries to `.aspect-video img` (the cell-thumbnail card container), preserving the original test intent without weakening it.
